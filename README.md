# 🍃 Studio Ghibli Theme

A cozy, earthy, and highly readable light theme for Visual Studio Code, inspired by the lush forests and comforting aesthetics of Studio Ghibli films.

![Theme Preview](https://raw.githubusercontent.com/YOUR-USERNAME/ghibli-forest-theme/main/screenshot.png)

## ✨ Features

- **Earthy Palette:** A harmonious blend of moss greens, bark browns, and soft terracotta clay on top of a creamy parchment background (`#FCF9F0`).
- **Syntax Highlighting:** Carefully tuned for excellent readability. It is perfect for reducing eye strain during long C++ competitive programming grinds or while debugging complex Python computer vision scripts.
- **Cozy UI:** Unobtrusive, minimalist interface colors with muted sage line numbers and vibrant, leaf-green string highlights.

## 📦 Installation

1. Open the **Extensions** view in VS Code (`Cmd + Shift + X` on Mac or `Ctrl + Shift + X` on Windows/Linux).
2. Search for `Studio-Ghibli Theme`.
3. Click **Install**.
4. Open the command palette (`Cmd + Shift + P` / `Ctrl + Shift + P`), select **Preferences: Color Theme**, and choose **Studio-Ghibli Theme**.

## 🎨 Recommended Settings

To get the absolute best, most immersive experience (as seen in the screenshot), add these settings to your VS Code `settings.json`:

```json
{
  "editor.fontFamily": "'Fantasque Sans Mono', Menlo, Monaco, monospace",
  "editor.cursorBlinking": "smooth",
  "editor.cursorSmoothCaretAnimation": "on",
  "editor.minimap.enabled": false,
  "breadcrumbs.enabled": false
}
```
